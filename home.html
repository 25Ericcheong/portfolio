Reviewer Hub Project

Reviewers Hub is a website that would allow users to write reviews on anything, users would be able to create their very own account and rate other reviews. Users would then be able to save certain reviews into their wishlist, write comments on reviewws, upvote or downvote reviews that are bad or good. All reviews, account created by users and reviews that are saved, would be stored in a database. The website utilizes google chart API to allow users to visualize what they have done and how the community is growing and a simple chatbot is implemented that has functionalities like allowing users to delete or ban comments or reviews or even find other existing users. Source code can be found here.

Robotic Design Project

The purpose of this project was to design a robot that is able to sort and deliver balls to two different designated containers. The robot will need to be able to segregate three different kinds of balls and will need to be able to start on the fly. It must also be able to transfer these balls as quickly as possible to the designated locations. The robot was programmed with the use of an Arduino, it was made of plywood and servos were used for any rotational movement.
